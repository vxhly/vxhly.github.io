    前言

    文件头是位于文件开头的一段承担一定任务的数据, 一般都在开头的部分。而文件尾对应的就在 文件的结束部分, 本篇将介绍常见的文件头和文件尾的十六进制数, 这些文件的文件头和文件尾几乎多是一样的。
