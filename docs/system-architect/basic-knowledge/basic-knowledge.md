## 指令系统

- 指令系统是计算机硬件的语言系统，是机器所具有的全部指令的集合，反映了计算机所拥有的基本功能。

## 计算机存储

- 存储速度从快到慢分别是：寄存器组、Cache、内存、Flash。

## 系统性能评价

- 通常用户采用评价程序来评价系统的性能。评价程序一般有专门的测量程序、仿真程序等，而评测准确度最高的评价程序是真实程序。
