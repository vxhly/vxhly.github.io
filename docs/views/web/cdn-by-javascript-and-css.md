::: tip 前言

收集常用的 JavaScript 库或者 CSS 库的 CDN 地址, 并且提供 JavaScript 在 window 对象中的全局名字。

:::

<!-- more -->

## vue@2.6.10

::: tip 全局名字

window 对象中的全局名字 `Vue`

:::

```text
https://cdn.jsdelivr.net/npm/vue@2.6.10/dist/vue.min.js
```
