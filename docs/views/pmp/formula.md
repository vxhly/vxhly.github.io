## 第六章 进度管理

### 三点估算

源自计划评审技术（PERT）。考虑估算中的不确定性和风险, 来提高估算的准确性。

三个估算值：最可能时间、最乐观时间、最悲观时间

::: tip 三角分布

期望值（平均值）= (最悲观 + 最乐观 + 最可能) / 3

:::

::: tip 贝塔分布 （默认分布）

期望值（平均值）= (最悲观 + 最乐观 + 最可能*4) / 6

:::

::: tip 标准差

标准差 = (最悲观 - 最乐观) / 6

:::
