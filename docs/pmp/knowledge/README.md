## 简介

国际项目管理师, 通常简称PMP, 是由美国国际项目管理学会组织所举办的项目管理领域资格认证, PMP采取资格审查与考试相结合的方式进行,
其理论体系主要依据为美国国际项目管理学会编写的《项目管理知识体系指南》
